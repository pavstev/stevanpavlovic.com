# Astro Project Components SDK

## Atoms (Foundational building blocks)

- **Analytics.astro**: GTM integration.
- **Avatar.astro**: User profile image with optimized `astro:assets`.
- **Badge.astro**: Small labels for tags, status, or categories.
- **Button.astro**: Primary interaction element with variants (ghost, glass, primary, etc.).
- **Container.astro**: Standardized page width wrapper.
- **DateTime.astro**: Formatted date display.
- **Grid.astro**: Responsive CSS Grid wrapper.
- **Heading.astro**: Styled typography for H1-H6.
- **Icon.astro**: Iconify integration.
- **Input.astro**: Styled text input.
- **Kbd.astro**: Keyboard shortcut indicator.
- **MetaData.astro**: SEO and OpenGraph head tags.
- **MeteorEffect.astro**: Animated background particles.
- **RollingNumbers.astro**: Count-up animation for metrics.
- **Separator.astro**: Visual divider line.
- **Slider.astro**: Range input.
- **SpotlightEffect.astro**: Mouse-tracking glow for cards.
- **Stack.astro**: Responsive Flexbox wrapper (Col/Row).
- **Switch.astro**: Toggle component.
- **Text.astro**: Base typography for paragraphs.
- **Tooltip.astro**: Interactive pop-over for citations or definitions.

## Molecules (Composite components)

- **AuthorBio.astro**: Compact profile card for post footers/sidebars.
- **BrandIdentity.astro**: Unified logo system with variants (`author`, `org`, `status`).
- **BriefSummary.astro**: Icon-prefixed description text.
- **CardImage.astro**: Optimized image wrapper for cards.
- **Chatbot.astro**: Local RAG-powered AI assistant.
- **DialogShell.astro**: Base styles for modals and palettes.
- **DisplayMenu.astro**: Grid/List view toggle.
- **GridTile.astro**: Layout cell for Bento-style grids.
- **LabelGroup.astro**: Horizontal list of badges.
- **LanguagePicker.astro**: i18n switcher.
- **ListActions.astro**: Pagination and filter controls.
- **MetaBar.astro**: Metadata dashboard for post headers.
- **MetricBlock.astro**: Labeled numeric stat with animation.
- **MobileNav.astro**: Navigation overlay for small screens.
- **PropertyField.astro**: Labeled value pair.

## Organisms (Complex sections)

- **ArticleRenderer.astro**: 12-column layout engine for content detail.
- **CompositionGrid.astro**: Unified page layout wrapper.
- **EnvironmentalBackground.astro**: Immersive animated backdrop.
- **OmniSearch.astro**: High-blur Command Palette (Spotlight-style).
- **ProgressNavigation.astro**: Auto-generated TOC with scroll tracking.
- **ResourceDiscovery.astro**: Collection engine for lists/grids.
- **ResumeEngine.astro**: Interactive and print-optimized professional profile.
- **StatsGrid.astro**: Responsive layout for high-level metrics.

## Layouts

- **Footer.astro**: Global site footer.
- **Header.astro**: Dynamic Island style navigation.
- **LayoutRoot.astro**: Base HTML document structure.
